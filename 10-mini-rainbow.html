This drill tests your ability to write functions accepting multiple arguments and then concatenating the return values of the function.

    You should write a function that generates the following HTML string.

    
    <div class="color {color-name}" style="color: ${hex}";>
      <span class="name">{color-name}</span>
      <span class="hex">{hex}</span>
      <span class="rgb">{rbg}</span>
    </div>
    

    Use this function to display 6 color divs inside the following `main` element. The colors are:

    - Red
    - Orange
    - Yellow
    - Green
    - Blue
    - Purple

    To use the function, it should accept three arguments. The string of the color name, the hex code and the rgb value.

    For example, using the function to generate the Yellow color div would be:

    
    nameOfFunction('Yellow', '#FFFF00', '255,255,0')
    

    Use Google to find the appropriate hex and rgb for each color.

    You do not need to write styles to color the elements unless you want to.
